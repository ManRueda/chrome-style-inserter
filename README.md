# chrome-style-inserter

With this chrome extension you can add styles dynamically when you debug a site with the dev-tools

###Versions

* 0.3.7:
    * CSS sanitize

* 0.3.6:
    * String parse error

* 0.3.5:
    * Grunt build
    * Minor fix

* 0.3.0:
    * LESS support
    * Style download
